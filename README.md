# E-commerce Price Monitoring Automation

A Python Selenium automation tool that monitors product prices from e-commerce websites and generates structured Excel reports.

## Features

- Automated browser scraping using Selenium
- Extracts product name, price, rating, and URL
- Exports CSV data
- Generates formatted Excel reports
- Designed for market research and price tracking

## Tech Stack

- Python
- Selenium
- Pandas
- OpenPyXL

## How to Run

```bash
pip install -r requirements.txt
python scraper.py
python report.py
